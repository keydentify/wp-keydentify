== Description ==
Lets your users secure their account with Keydentify.

== Installation ==
1. Unzip and upload `/keydentify/` to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activate your website on https://www.keydentify.me and get your credentials
4. Put your credentials in Keydentify admin
5. Activate Keydentify for your account, disconnect and reconnect
